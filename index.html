Printables Project TinkerCAD Project

What's cool

I love my System76 Launch keyboard, but I don't love the ‘daylight’ keycaps. I wanted some ‘pudding’ style keycaps, with black tops and translucent bodies & legends. Due to the atypical sizes of keys used on the Launch, most retail keycaps don't have the proper sizes for all the meta-keys (from Shifts, Ctrl/Alt/etc…, to the truly bizarre 1.5u Enter key)… so I decided to print them myself.

Built on a solid foundation

I found the awesome KeyV2 library for openscad, and was able to generate XDA key profiles with the rounded cherry stems, and the appropriate stem height & key overhang to match the stock keys.

Custom tooling

I used TinkerCAD to apply FontAwesome Pro icons as the legends, in a multi-material design, and export STLs that PrusaSlicer was happy with.

Multi-material printing

The key models are upside-down to get the cleanest print on layer 1, and are therefore flat on the top instead of dished; I like how this further helps my fingers distinguish when I'm doing operations instead of typing (and gives me a tactile warning when I've moved off-center).

Perfect layout

Keys are 1u (standard with) unless noted. I use right-alt as my compose key, so that's included. If you want to try out my layout on your Launch, the layout file is available in the github repo, along with other sources.

FAQs

How did you print?

This is intended to be printed on a multi-material printer; I use a Mk3S+ MMU2S. In the 3MFG & GCODE files, extruder 2 is white, extruder 3 is black, and extruder 5 is transparent, and the nozzle size is 0.25mm for those detailed icons. Only the first layer has the white legend; subsequent layers are printed in the body material (clear/transparent). This provides the best balance of daytime visibility, with a high-contrast between the face & legend, and plenty of translucency so the LEDs can show through.

The actual keys in the photos are in daily use on my keyboard, printed in Prusament PETG on a textured plate. 

Which keys are included?

This is not a full set of keys… for the standard alphanumeric & function keys, you're better off buying some factory-made keycaps (the photos feature LTC XDA-profile Lava Caps in Pudding Black). The keys included here are:

  • Caps Lock 1.5u

  • Shift 2u, 1.5u

  • Ctrl/Cmd 1.5u

  • Meta/Super/Windows

  • Menu

  • Alt

  • Space 2u

  • Compose

  • Layer 2

  • Layer 3 1.5u

  • Enter 1.5u

  • Del

  • Bksp

  • Home + Play/Pause

  • PgUp + Vol Up

  • PgDn + Vol Down

  • End + Mute
